## Exam Objectives

The eJPT v2 exam evaluates your ability to perform practical penetration testing tasks, including:

- Locating endpoints within a network  
- Identifying open ports and running services  
- Determining the operating system of a target  
- Extracting company and user information from public sources  
- Gathering technical information via passive and active reconnaissance  
- Enumerating network, system, and user information from targets  
- Identifying vulnerabilities and assessing their impact  
- Transferring files to and from compromised systems  
- Extracting hash and password information  
- Identifying, modifying, and using public exploits  
- Conducting exploitation using Metasploit  
- Performing pivoting via routing and port forwarding  
- Conducting brute-force password attacks and hash cracking  
- Identifying and exploiting web application vulnerabilities  
- Locating hidden files and directories  
- Conducting web application reconnaissance

## eJPT Exam Overview

- **Time limit:** 48 hours (lab access enabled for the full duration)  
- **Questions:** 35  
- **Certification validity:** 3 years  

### Exam Domains & Weighting

**Assessment Methodologies Domain (25%)**
- Identify open ports and services  
- Evaluate vulnerability impact and criticality  
- Minimum score: 90%

**Host & Network Auditing Domain (25%)**
- Enumerate systems and network information  
- Extract hash and password data  
- Minimum score: 80%

**Host & Network Penetration Testing Domain (35%)**
- Conduct brute-force attacks  
- Perform exploitation using Metasploit  
- Minimum score: 70%

**Web Application Penetration Testing Domain (15%)**
- Exploit web application vulnerabilities  
- Locate hidden files and directories  
- Minimum score: 60%

**Overall requirement:**  
At least **70% total score**, while meeting minimum domain thresholds.

## Exam Philosophy

From a technical perspective, the eJPT exam simulates a **black-box penetration test**:

- minimal initial information about the target network,
- browser-based labs (no VPN required),
- unrestricted access to the lab environment for the full exam duration,
- a provided Letter of Engagement,
- hands-on, open-book format,
- dynamic, randomly generated flags.

The exam evaluates your ability to **find answers**, not follow a rigid path. Tooling, methodology, and workflow decisions are intentionally left to the candidate.
