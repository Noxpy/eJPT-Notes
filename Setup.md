### Linux Steps to Clone a GitHub Repository

1. **Install Git (if not already installed)**  
   Run the following command to install Git on your system:

   ```bash
   sudo apt update
   sudo apt install git
   ```

2. **Navigate to the directory**  
   Use `cd` to navigate to the directory where you want to clone the repository:

   ```bash
   cd /path/to/your/directory
   ```

3. **Clone the repository**  
   Run the following command to clone the eJPT-Notes repository:

   ```bash
   git clone https://github.com/Noxpy/eJPT-Notes.git
   ```

4. **Navigate to the cloned repository**  
   After cloning, go into the directory:

   ```bash
   cd eJPT-Notes
   ```

### Windows Steps to Clone a GitHub Repository

1. **Install Git (if not already installed)**  
   Download and install Git from [here](https://git-scm.com/download/win). Follow the installation prompts, keeping the default settings.

2. **Open Command Prompt or PowerShell**  
   Press `Win + R`, type `cmd` or `powershell`, and press Enter.

3. **Navigate to the directory**  
   Use `cd` to navigate to the directory where you want to clone the repository:

   ```bash
   cd C:\path\to\your\directory
   ```

4. **Clone the repository**  
   Run the following command to clone the eJPT-Notes repository:

   ```bash
   git clone https://github.com/Noxpy/eJPT-Notes.git
   ```

5. **Navigate to the cloned repository**  
   Once cloned, enter the directory:

   ```bash
   cd eJPT-Notes
   ```
